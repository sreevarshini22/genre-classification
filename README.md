# Movie Genre Classification using Machine Learning

## Project Overview

This project focuses on building a Machine Learning model that predicts the genre of a movie based on its plot summary or textual description. The system analyzes the text data and classifies it into the appropriate movie genre.

## Objective

The objective of this project is to apply Natural Language Processing (NLP) and Machine Learning techniques to automatically classify movies into genres using their plot summaries.

## Technologies Used

* Python
* Google Colab
* Scikit-learn
* Pandas
* NumPy
* Natural Language Processing (NLP)

## Machine Learning Techniques

The following techniques are used in this project:

* **Text Preprocessing**

  * Removing stop words
  * Lowercasing
  * Cleaning text data

* **Feature Extraction**

  * TF-IDF Vectorization

* **Classification Algorithms**

  * Naive Bayes
  * Logistic Regression
  * Support Vector Machine (SVM)

## Dataset
The dataset files are provided in compressed (.zip) format.
Extract the files before running the notebook in Google Colab.


Files used:

* train_data.txt
* test_data.txt
* test_data_solution.txt

## Project Workflow

1. Load the dataset
2. Perform text preprocessing
3. Convert text into numerical features using TF-IDF
4. Train machine learning models
5. Evaluate model performance
6. Predict movie genre based on new plot summaries

## Expected Output

The trained model predicts the most suitable genre for a movie based on the given plot summary.
